# 튜토리얼 08: 심화 워크플로우와 자동화

> 난이도: 고급 | 소요시간: 30분

## 학습 목표

- 여러 플러그인과 MCP 서버를 조합하여 복합 분석하기
- 실전 영업 시나리오에서 End-to-End 워크플로우 실행하기
- 경쟁사 분석, 투자 리포트 생성 등 고급 활용법 익히기

## 워크플로우 1: 신규 고객 발굴부터 계약까지

### 시나리오
우리팀(코스콤)이 "카카오뱅크"에 금융 데이터 분석 플랫폼을 영업하려 합니다.

### Step 1: 기업 리서치 + 재무 분석

```
카카오뱅크에 대해 종합 리서치를 해줘:

1. 기업 개요: 사업 모델, 주요 서비스, 경영진
2. 주가 분석: 최근 주가 동향, 기술적 지표(RSI, MACD)
3. 재무 분석: 최근 실적, 매출 성장률, 수익성
4. 최신 뉴스: 디지털 혁신, IT 투자 관련 뉴스
5. 경쟁 환경: 토스뱅크, 케이뱅크와의 비교

영업 관점에서 어떤 포인트를 공략해야 할지 분석해줘.
```

### Step 2: 경쟁사 분석

```
/sales:competitive-intelligence
```

입력:

```
코스콤의 금융 데이터 분석 플랫폼과 경쟁하는 업체들을 분석해줘:

우리 제품: 실시간 시세 데이터 + AI 분석 + 리스크 관리
경쟁사:
1. Bloomberg Terminal
2. Refinitiv (LSEG)
3. FactSet
4. 한국투자증권 자체 시스템

각 경쟁사 대비 우리의 강점, 약점, 차별화 포인트를 정리해줘.
```

### Step 3: 맞춤형 제안서 자료 생성

```
/sales:create-an-asset
```

입력:

```
카카오뱅크 대상 영업 제안서 자료를 만들어줘:

대상: 카카오뱅크 데이터사이언스팀
우리 제품: 금융 데이터 분석 플랫폼 (코스콤)
핵심 가치:
- 실시간 국내외 시세 데이터 (지연 < 100ms)
- AI 기반 리스크 분석 엔진
- 맞춤형 대시보드 & API
- 금융보안원 인증, ISMS-P 획득

고객 페인포인트:
- 인터넷전문은행이라 자체 인프라 구축 부담
- 빠른 시장 대응 필요
- 규제 준수 (바젤 III, 리스크 관리)

One-pager 형태로 작성해줘.
```

### Step 4: 콜 준비 + 미팅

```
/sales:call-prep
```

입력:

```
카카오뱅크 데이터사이언스팀과 첫 미팅 준비:
- 일시: 다음 주 화요일 14:00
- 참석자: 이팀장, 김과장 (데이터사이언스팀)
- 장소: 카카오뱅크 판교 오피스
- 목표: 제품 소개 및 POC 제안

미팅 자료에 카카오뱅크 주가/실적 데이터도 포함해줘.
```

### Step 5: 미팅 후 팔로업

```
/sales:call-summary
```

입력:

```
카카오뱅크 미팅 결과 정리:

참석자: 이팀장, 김과장 / 코스콤 박부장, 최대리
논의 내용:
- 현재 자체 데이터 파이프라인 운영 중이나 유지보수 부담 큼
- 실시간 해외 시세 데이터 니즈 확인 (특히 미국 주식)
- AI 리스크 분석에 높은 관심
- 보안 인증 관련 상세 자료 요청
- 2주 내 POC 환경 제공 가능 여부 확인 요청
- 예산은 연간 5억 내외로 검토 가능
- 경쟁사 B사도 제안 중이라 함

액션 아이템, 팔로업 이메일, 내부 보고서를 생성해줘.
```

## 워크플로우 2: 분기 실적 분석 리포트

### 시나리오
분기 영업 실적을 정리하고 경영진 보고 자료를 준비합니다.

### Step 1: 파이프라인 현황 정리

```
/sales:pipeline-review
```

입력:

```
2025년 4분기 파이프라인 리뷰:

완료 건:
1. 삼성SDS - 클라우드 인프라 - 50억 - Won
2. KB국민은행 - 데이터 플랫폼 - 30억 - Won
3. 미래에셋 - API 연동 - 8억 - Lost (경쟁사)

진행 중:
4. 카카오뱅크 - 데이터 분석 - 5억 - POC 단계 (60%)
5. 신한은행 - AI 트레이딩 - 40억 - 제안 단계 (30%)
6. NH투자증권 - 리스크 관리 - 25억 - 협상 단계 (70%)
7. 하나은행 - API 게이트웨이 - 15억 - 발굴 단계 (15%)

분석 및 주간 액션 플랜을 생성해줘.
```

### Step 2: 매출 예측 생성

```
/sales:forecast
```

입력:

```
위 파이프라인 데이터를 기반으로 다음 분기(2026 Q1) 매출 예측을 해줘:

Best / Likely / Worst 시나리오
Commit vs Upside 구분
갭 분석: 분기 목표 80억 대비
```

### Step 3: 재무 성과 리포트

```
/finance:financial-statements
```

입력:

```
2025년 4분기 팀 실적 손익계산서를 작성해줘:

매출: 88억 (삼성SDS 50억 + KB국민은행 30억 + 기타 8억)
매출원가: 52억 (인건비 35억 + 인프라 비용 12억 + 기타 5억)
판관비: 15억 (마케팅 5억 + 영업활동비 3억 + 일반관리 7억)

전분기(Q3) 데이터: 매출 65억, 매출원가 40억, 판관비 12억

전분기 대비 변동 분석도 포함해줘.
```

### Step 4: 시각화 대시보드

```
/data:build-dashboard
```

입력:

```
경영진 보고용 분기 실적 대시보드를 만들어줘:

차트 1: 분기별 매출 추이 (Q1~Q4) - 바차트
차트 2: 파이프라인 퍼널 (발굴→제안→협상→완료) - 퍼널
차트 3: 고객사별 매출 비중 - 도넛차트
차트 4: 목표 대비 달성률 - 게이지 차트
차트 5: 영업사원별 실적 - 가로 바차트

데이터:
- Q1: 45억, Q2: 55억, Q3: 65억, Q4: 88억 (목표 80억)
- 파이프라인: 발굴 2건/20억, 제안 1건/40억, 협상 1건/25억, 완료 2건/80억
```

## 워크플로우 3: 시장 인텔리전스 리포트

### 시나리오
주간 시장 동향 리포트를 자동 생성합니다.

### Step 1: 글로벌 시장 동향

```
다음 데이터를 조회하고 주간 시장 동향 리포트를 작성해줘:

1. 미국 주요 지수 동향: S&P 500 관련 ETF(SPY), 나스닥(QQQ), 다우(DIA)
2. 빅테크 주간 실적: AAPL, MSFT, GOOGL, AMZN, NVDA, META, TSLA
3. 원자재: 금, 은, WTI 원유, 천연가스
4. 환율: USD/KRW, USD/JPY, EUR/USD
5. 경제지표: 미국 CPI, 실업률, 금리

각 항목의 주간 변동과 주요 이슈를 요약해줘.
```

### Step 2: 금융 섹터 심층 분석

```
다음 글로벌 금융 기업들의 재무 데이터를 비교 분석해줘:
- JPMorgan (JPM)
- Goldman Sachs (GS)
- Morgan Stanley (MS)
- Bank of America (BAC)

비교 항목:
1. 매출 성장률 (YoY)
2. 순이익률
3. ROE
4. PER / PBR
5. 배당수익률

우리 고객사(한국 금융기관)에게 시사하는 바를 정리해줘.
```

### Step 3: 뉴스 기반 기회 발굴

```
다음 키워드로 최신 뉴스를 검색하고 영업 기회를 발굴해줘:

키워드:
- 한국 금융 디지털 전환
- 금융권 AI 도입
- 금융 데이터 분석 플랫폼
- 오픈뱅킹 API

각 뉴스에서 잠재 영업 기회를 식별하고
타겟 고객, 접근 전략, 예상 규모를 정리해줘.
```

### Step 1: 고객 포트폴리오 분석

```
우리 주요 고객사들의 현재 상태를 종합 분석해줘:

| 고객 | 계약규모 | 만기 | 건강도 |
|------|---------|------|--------|
| 삼성SDS | 50억/년 | 2026.06 | 양호 |
| KB국민은행 | 36억/년 | 2026.03 | 주의 (장애 이슈) |
| NH투자증권 | 25억/년 | 2025.12 | 갱신 협상 중 |
| 하나은행 | 15억/년 | 2026.09 | 양호 |

각 고객별:
1. 주가/재무 동향 (투자 여력 파악)
2. 최근 뉴스 (조직 변화, IT 투자 계획)
3. 계약 갱신 리스크
4. 업셀링/크로스셀링 기회
```

### Step 2: 이탈 방지 전략

```
KB국민은행 계약 갱신이 3월에 예정되어 있고,
최근 장애 이슈로 불만이 있습니다.

이탈 방지 전략을 수립해줘:
1. 장애 재발 방지 계획
2. 추가 가치 제안 (무상 업그레이드 등)
3. 관계 회복 커뮤니케이션 플랜
4. 경쟁사 전환 시 고객의 전환 비용 분석
```

## 워크플로우 5: 투자 분석 리포트

### 시나리오
고객사 임원에게 제공할 시장 인사이트 리포트를 생성합니다.

```
고객사 임원 대상 '2026년 1분기 AI 반도체 시장 전망' 리포트를 작성해줘:

다음 데이터를 조회해서 분석에 활용해줘:
1. NVDA, AMD, INTC, AVGO, QCOM 주가 및 재무 데이터
2. 각 기업의 기술적 지표 (RSI, MACD)
3. 실적 전망 (EPS 추정치)
4. 관련 뉴스 및 감성 분석

리포트 구조:
1. Executive Summary
2. 시장 개요 및 트렌드
3. 주요 기업 분석 (표 형태)
4. 기술적 분석 요약
5. 투자 시사점
6. 리스크 요인

전문적이면서도 읽기 쉽게 작성해줘.
```

## 자주 사용하는 복합 명령 패턴

### 패턴 1: 리서치 + 분석 + 자료 생성

```
[기업명]에 대해 리서치하고 주가/재무 분석 후 영업 제안서를 만들어줘
```

### 패턴 2: 시장 데이터 + 비교 분석 + 대시보드

```
[종목들]의 주가/재무 데이터를 비교 분석하고 대시보드로 만들어줘
```

### 패턴 3: 고객 문의 → 조사 → 응답 → 문서화

```
[이슈 설명] → 원인 조사 → 고객 응답 작성 → KB 문서로 정리
```

### 패턴 4: 파이프라인 → 예측 → 리포트 → 시각화

```
현재 파이프라인 분석 → 매출 예측 → 실적 리포트 → 대시보드 생성
```

## 핵심 팁

1. **맥락 유지**: 한 세션에서 연속 작업하면 이전 맥락이 유지되어 더 일관된 결과를 얻습니다
2. **데이터 결합**: MCP 실시간 데이터 + 플러그인 분석 기능을 조합하면 강력합니다
3. **반복 자동화**: 매주/매월 반복하는 리포트는 동일한 프롬프트를 저장해두세요
4. **품질 검증**: 중요한 분석 결과는 `/data:validate`로 반드시 검증하세요
