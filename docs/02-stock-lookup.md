# 튜토리얼 02: 주식 정보 조회 기초

> 난이도: 초급 | 소요시간: 15분

## 학습 목표

- Alpha Vantage MCP로 다양한 주식 데이터 조회하기
- Financial Datasets MCP로 기업 재무 데이터 조회하기
- 주가, 거래량, 기본적인 기업 정보 읽는 법 이해하기

## 실습 1: 실시간 시세 조회

### 1-1. 미국 주식 시세

```
Microsoft 현재 주가를 조회해줘
```

결과에 포함되는 항목:
- **price**: 현재가
- **open/high/low**: 시가/고가/저가
- **volume**: 거래량
- **change / changePercent**: 전일 대비 변동

### 1-2. 여러 종목 한번에 조회

```
AAPL, MSFT, GOOGL, AMZN, NVDA 주가를 한눈에 비교해줘
```

### 1-3. 시장 상태 확인

```
현재 전 세계 주요 거래소의 개장/폐장 상태를 알려줘
```

## 실습 2: 주가 시계열 데이터

### 2-1. 일간 시계열

```
Tesla의 최근 한 달간 일별 주가 추이를 보여줘
```

### 2-2. 주간 시계열

```
NVDA의 최근 6개월 주간 주가를 분석해줘
```

### 2-3. 월간 시계열

```
Apple의 최근 1년간 월별 주가 흐름을 정리해줘
```

## 실습 3: 기업 정보 조회

### 3-1. 기업 개요

```
AAPL 기업 개요를 상세하게 보여줘. 시가총액, PER, 배당수익률 포함해서
```

결과에 포함되는 주요 지표:
- **MarketCap**: 시가총액
- **PERatio**: 주가수익비율
- **DividendYield**: 배당수익률
- **52WeekHigh/Low**: 52주 최고/최저

### 3-2. 배당 이력

```
Microsoft의 배당 이력을 보여줘
```

### 3-3. 주식 분할 이력

```
NVDA의 주식분할 이력을 조회해줘
```

## 실습 4: Financial Datasets 활용

### 4-1. 손익계산서

```
Apple의 최근 4분기 손익계산서를 보여줘
```

### 4-2. 대차대조표

```
Tesla의 최근 연도별 대차대조표를 분석해줘
```

### 4-3. 현금흐름표

```
Microsoft의 현금흐름표에서 영업활동, 투자활동, 재무활동 현금흐름을 비교해줘
```

## 실습 5: 뉴스와 감성 분석

### 5-1. 기업 뉴스 조회

```
NVDA 관련 최신 뉴스와 시장 감성을 분석해줘
```

### 5-2. 미국 시장 상위/하위 종목

```
오늘 미국 시장 상승률 TOP 10, 하락률 TOP 10 종목을 알려줘
```

## 주요 명령어 정리

| 용도 | 예시 프롬프트 |
|------|-------------|
| 현재가 | "AAPL 현재 주가" |
| 일간 추이 | "TSLA 최근 한 달 일별 주가" |
| 기업 개요 | "MSFT 기업 개요, PER, 시가총액" |
| 재무제표 | "AAPL 손익계산서 보여줘" |
| 배당 | "MSFT 배당 이력" |
| 뉴스 | "NVDA 관련 뉴스와 감성분석" |
| 시장 현황 | "미국 시장 상승률/하락률 TOP 10" |

## 알아두면 좋은 팁

1. **티커 심볼**: 미국 주식은 영문 티커 (AAPL, MSFT 등)를 사용합니다
2. **한국 주식**: 직접 KRX 조회가 안되므로 웹 검색을 조합합니다
3. **API 제한**: 무료 티어는 하루 500콜, 분당 5콜 제한이 있습니다
4. **장 마감 후**: 마지막 거래일 종가가 표시됩니다
