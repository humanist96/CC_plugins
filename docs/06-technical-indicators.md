## 실습 1: 이동평균 분석

### 1-1. 골든크로스 / 데드크로스 확인

```
AAPL의 20일, 60일, 120일 이동평균선(SMA)을 조회하고
골든크로스나 데드크로스가 발생했는지 확인해줘
```

### 1-2. EMA vs SMA 비교

```
NVDA의 20일 SMA와 20일 EMA를 비교해줘.
최근 한 달간 두 지표의 차이와 현재가 대비 위치를 분석해줘.
```

### 1-3. VWAP 분석

```
MSFT의 장중 VWAP(거래량가중평균가)을 조회해줘.
현재 주가가 VWAP 대비 어디에 위치하는지 분석해줘.
```

## 실습 2: 모멘텀 분석

### 2-1. RSI 분석

```
Tesla(TSLA)의 14일 RSI를 조회하고 분석해줘.
- 현재 RSI 값
- 과매수(70 이상) / 과매도(30 이하) 여부
- 최근 한 달간 RSI 추이
- 다이버전스 발생 여부 (주가와 RSI 방향 불일치)
```

### 2-2. MACD 분석

```
AAPL의 MACD를 조회해줘 (설정: 12, 26, 9).
- MACD 라인과 시그널 라인 값
- MACD 히스토그램 방향
- 매수/매도 시그널 여부
- 최근 크로스오버 시점
```

### 2-3. 스토캐스틱 분석

```
GOOGL의 스토캐스틱 오실레이터를 조회해줘.
%K와 %D의 현재 위치, 교차 시그널을 분석해줘.
```

### 2-4. ADX(추세 강도) 분석

```
AMZN의 ADX를 조회해서 현재 추세의 강도를 판단해줘.
- ADX > 25: 강한 추세
- ADX < 20: 추세 없음 (횡보)
+DI와 -DI도 함께 분석해줘.
```

## 실습 3: 변동성 분석

### 3-1. 볼린저밴드 분석

```
NVDA의 볼린저밴드(20일, 2 표준편차)를 조회하고 분석해줘.
- 상단밴드, 중간밴드, 하단밴드 값
- 밴드폭(Band Width) - 변동성 수준
- 현재가의 밴드 내 위치 (%B)
- 밴드 스퀴즈(수축) 여부 - 큰 움직임 예고 가능
```

### 3-2. ATR 분석

```
TSLA의 14일 ATR(평균진폭)을 조회해줘.
현재 ATR 값을 기반으로:
- 일간 예상 변동 범위
- 손절 라인 설정 가이드 (2x ATR)
- 최근 ATR 추이 (변동성 확대/축소)
```

### 3-3. 파라볼릭 SAR

```
AAPL의 파라볼릭 SAR를 조회해줘.
현재 SAR 방향(상승/하락)과 추세 전환 시점을 분석해줘.
```

## 실습 4: 거래량 분석

### 4-1. OBV (On Balance Volume)

```
MSFT의 OBV를 조회하고 주가와 비교 분석해줘.
OBV가 상승하면서 주가가 횡보하면 상승 잠재력 있음.
OBV가 하락하면서 주가가 유지되면 하락 위험 있음.
```

### 4-2. 자금흐름지수 (MFI)

```
NVDA의 MFI(14일)를 조회해줘.
RSI와 비슷하지만 거래량을 포함한 지표야.
과매수(80+)/과매도(20-) 여부를 확인해줘.
```

## 실습 5: 종합 기술적 분석

### 5-1. 단일 종목 종합 분석

```
Apple(AAPL)에 대해 다음 지표를 모두 조회하고 종합 분석 리포트를 작성해줘:

1. 추세: SMA(20, 60, 120), EMA(20)
2. 모멘텀: RSI(14), MACD(12,26,9), 스토캐스틱
3. 변동성: 볼린저밴드(20,2), ATR(14)
4. 거래량: OBV, MFI

종합 판단:
- 현재 추세 방향과 강도
- 매수/매도 시그널 요약
- 주요 지지선/저항선
- 단기 전망
```

### 5-2. 비교 기술적 분석

```
AAPL, MSFT, GOOGL 3개 종목의 RSI, MACD, 볼린저밴드를 비교해서
현재 기술적으로 가장 매력적인 종목을 분석해줘.

비교 기준:
- 과매도 영역 근접 (저가 매수 기회)
- 추세 전환 시그널
- 변동성 수준
```

### 5-3. 섹터 분석

```
미국 빅테크 5개 종목(AAPL, MSFT, GOOGL, AMZN, NVDA)의
ADX로 추세 강도를 비교하고,
RSI로 과매수/과매도 상태를 한눈에 정리해줘.
```

## 실습 6: 분석 고급 기법

### 6-1. 고급 분석 윈도우

```
NVDA에 대해 고정 윈도우 분석(ANALYTICS_FIXED_WINDOW)을 실행해줘.
최근 3개월 데이터로 수익률, 변동성, 샤프비율을 계산해줘.
```

### 6-2. 슬라이딩 윈도우 분석

```
AAPL에 대해 30일 슬라이딩 윈도우로
이동 평균 수익률과 이동 변동성의 추이를 분석해줘.
```

## 지표 해석 가이드

### RSI 해석
| RSI 값 | 해석 |
|---------|------|
| 70+ | 과매수 - 하락 반전 가능 |
| 50~70 | 상승 추세 |
| 30~50 | 하락 추세 |
| 30 이하 | 과매도 - 반등 가능 |

### MACD 시그널
| 상황 | 해석 |
|------|------|
| MACD > 시그널 (상향 돌파) | 매수 시그널 |
| MACD < 시그널 (하향 돌파) | 매도 시그널 |
| 히스토그램 양(+) 확대 | 상승 모멘텀 강화 |
| 히스토그램 음(-) 확대 | 하락 모멘텀 강화 |

### 볼린저밴드 해석
| 상황 | 해석 |
|------|------|
| 주가 > 상단밴드 | 과매수, 조정 가능 |
| 주가 < 하단밴드 | 과매도, 반등 가능 |
| 밴드 수축 (스퀴즈) | 큰 움직임 임박 |
| 밴드 확대 | 변동성 증가 중 |
