# 튜토리얼 01: 시작하기

> 난이도: 초급 | 소요시간: 10분

## 학습 목표

- Claude Code 기본 사용법 익히기
- 설치된 플러그인과 MCP 서버 확인하기
- 첫 번째 명령어 실행하기

## 사전 준비

Claude Code가 설치되어 있어야 합니다.

```bash
# Claude Code 실행
claude
```

## 실습 1: 환경 확인

### 1-1. MCP 서버 상태 확인

Claude Code 프롬프트에서 다음을 입력하세요:

```
/mcp
```

Alpha Vantage와 Financial Datasets 서버가 표시되면 정상입니다.

### 1-2. 설치된 플러그인 확인

```
설치된 플러그인 목록을 보여줘
```

다음 플러그인들이 보여야 합니다:
- sales (영업)
- finance (재무)
- data (데이터 분석)
- customer-support (고객 지원)
- enterprise-search (통합 검색)
- productivity (생산성)

## 실습 2: 첫 번째 주가 조회

### 2-1. 간단한 주가 조회

```
AAPL 현재 주가 알려줘
```

Claude가 Alpha Vantage MCP를 통해 Apple 주가를 조회하고 결과를 보여줍니다.

### 2-2. 한국 주식 조회

```
삼성전자 주가 알려줘
```

> 참고: Alpha Vantage는 한국 거래소(KRX) 직접 조회를 지원하지 않습니다.
> 대신 런던 거래소(SMSN.LON) 데이터와 웹 검색을 조합하여 정보를 제공합니다.

### 2-3. 환율 조회

```
USD/KRW 현재 환율 조회해줘
```

## 실습 3: 첫 번째 플러그인 명령어

### 3-1. 생산성 대시보드 시작

```
/start
```

productivity 플러그인의 대시보드가 초기화됩니다.

### 3-2. 간단한 데이터 분석 요청

```
오늘 코스피 상위 종목들의 동향을 분석해줘
```

## 핵심 정리

| 기능 | 명령어/입력 |
|------|------------|
| MCP 상태 확인 | `/mcp` |
| 주가 조회 | "AAPL 주가 알려줘" |
| 환율 조회 | "USD/KRW 환율 조회해줘" |
| 대시보드 | `/start` |
