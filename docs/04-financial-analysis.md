# 튜토리얼 04: 재무 분석 플러그인

> 난이도: 중급 | 소요시간: 25분

## 학습 목표

- Finance 플러그인으로 재무제표 생성 및 분석하기
- 분개장 작성, 계정 대조 실습하기
- 예산 대비 실적 분석(Variance Analysis) 수행하기

## Finance 플러그인 명령어 총정리

| 명령어 | 기능 |
|--------|------|
| `/finance:financial-statements` | 재무제표 생성 (손익계산서, 대차대조표, 현금흐름표) |
| `/finance:journal-entry-prep` | 분개장 작성 |
| `/finance:reconciliation` | 계정 대조 |
| `/finance:variance-analysis` | 예산 대비 실적 분석 |
| `/finance:close-management` | 월말 결산 관리 |
| `/finance:audit-support` | SOX 감사 지원 |

## 실습 1: 재무제표 분석

### 1-1. 손익계산서 생성

```
/finance:financial-statements
```

입력:

```
다음 데이터로 2025년 4분기 손익계산서를 생성해줘.

매출액: 150억
매출원가: 90억
판관비: 30억
영업외수익: 2억
이자비용: 3억
법인세율: 22%

전분기 비교 포함해줘:
3분기 매출액: 140억, 매출원가: 85억, 판관비: 28억
```

### 1-2. 실제 기업 재무제표 조회

Alpha Vantage와 결합하여 실제 데이터를 분석해봅시다:

```
Apple(AAPL)의 최근 4분기 손익계산서를 조회하고, 매출 성장률과 영업이익률 추이를 분석해줘
```

### 1-3. 대차대조표 분석

```
Tesla(TSLA)의 최근 대차대조표를 조회해서 유동비율, 부채비율, 자기자본이익률(ROE)을 계산하고 재무 건전성을 평가해줘
```

## 실습 2: 분개장 작성

### 시나리오
월말 결산을 위한 주요 분개를 작성합니다.

### 2-1. 매출 인식 분개

```
/finance:journal-entry-prep
```

입력:

```
다음 거래의 분개를 작성해줘:

1. 12월 매출 인식
   - 고객사 A에 금융 데이터 플랫폼 라이선스 판매: 5억원
   - 계약 기간: 12개월, 선수금으로 전액 수령 완료
   - 수익인식 기준: 기간 배분 (월할)

2. 12월 급여 지급
   - 총 급여: 8억원
   - 4대보험 회사부담금: 7,200만원
   - 소득세 원천징수: 1.2억원

3. 감가상각
   - 서버 장비 취득가: 10억원
   - 내용연수: 5년, 정액법
   - 당월 상각액 계산 필요
```

### 2-2. 결산 조정 분개

```
다음 결산 조정 분개를 작성해줘:

1. 미지급비용 인식
   - 12월 전기료 미수령: 2,500만원
   - 12월 통신비 미수령: 800만원

2. 선급비용 상각
   - 보험료 연간 선납: 6,000만원 (1월~12월)
   - 12월분 비용화 필요

3. 대손충당금 설정
   - 매출채권 잔액: 30억원
   - 대손율: 2%
   - 기존 충당금: 4,000만원
```

## 실습 3: 계정 대조

### 3-1. 은행 잔액 대조

```
/finance:reconciliation
```

입력:

```
12월 31일 기준 은행 잔액 대조를 해줘:

장부 잔액: 15억 3,200만원
은행 잔액: 15억 8,500만원

불일치 항목:
- 미결제 수표: 3,800만원 (12/28 발행, 1/3 결제)
- 은행 이체 수수료: 50만원 (장부 미반영)
- 이자 수입: 120만원 (장부 미반영)
- 고객 입금: 1,500만원 (은행 미반영 - 12/31 저녁 입금)
- 부도 수표: 800만원 (장부 미반영)
```

### 3-2. 매출채권 대조

```
매출채권 보조원장과 총계정원장을 대조해줘:

총계정원장 매출채권 잔액: 45억원

보조원장 내역:
- 삼성SDS: 15억원 (11월 매출, 만기 1/15)
- KB국민은행: 12억원 (12월 매출, 만기 2/15)
- 하나은행: 8억원 (10월 매출, 만기 12/31 - 연체)
- NH투자증권: 7억원 (12월 매출, 만기 1/31)
- 기타: 2.5억원

보조원장 합계: 44.5억원
차이 원인을 분석하고 조정 방안을 제시해줘
```

## 실습 4: 예산 대비 실적 분석

### 4-1. 부서별 예산 분석

```
/finance:variance-analysis
```

입력:

```
2025년 4분기 부서별 예산 대비 실적을 분석해줘:

| 항목 | 예산 | 실적 |
|------|------|------|
| 매출액 | 150억 | 162억 |
| 매출원가 | 90억 | 95억 |
| 인건비 | 25억 | 27억 |
| 마케팅비 | 5억 | 8억 |
| 연구개발비 | 10억 | 9억 |
| 일반관리비 | 8억 | 7.5억 |
| 영업이익 | 12억 | 15.5억 |

주요 변동 원인:
- 매출: 삼성SDS 대형 프로젝트 조기 수주
- 인건비: 4분기 개발인력 3명 추가 채용
- 마케팅비: 연말 프로모션 캠페인 집행

워터폴 차트 형태로 분석 결과를 정리해줘
```

### 4-2. 전년 동기 대비 분석

```
다음 데이터로 전년 동기 대비(YoY) 분석을 해줘:

2024년 4분기: 매출 120억, 영업이익 8억, 순이익 6억
2025년 4분기: 매출 162억, 영업이익 15.5억, 순이익 12억

매출 성장률, 영업이익률 변화, 순이익률 변화를 분석하고
성장 요인과 주의할 점을 정리해줘
```

## 실습 5: 월말 결산 관리

### 5-1. 결산 체크리스트 생성

```
/finance:close-management
```

입력:

```
12월 결산 일정을 관리해줘:

결산 마감일: 1월 10일

주요 업무:
- Day 1-2: 매출/매입 마감, 재고 실사
- Day 3-4: 미지급비용 계상, 선급/선수 정리
- Day 5: 감가상각, 대손충당금
- Day 6-7: 계정 대조 및 조정
- Day 8: 재무제표 초안 작성
- Day 9: 경영진 검토
- Day 10: 최종 마감

담당자: 김회계(매출), 이재무(매입), 박결산(조정)
```

## 실습 6: MCP 데이터와 결합 분석

### 6-1. 경쟁사 재무 비교

```
Apple, Microsoft, Google의 최근 분기 재무제표를 조회해서
매출 성장률, 영업이익률, ROE를 비교 분석해줘.
표와 함께 각 기업의 재무적 강점과 약점을 정리해줘.
```

### 6-2. 산업 평균 대비 분석

```
NVDA의 재무제표를 조회하고, 반도체 산업 평균과 비교하여
수익성, 안정성, 성장성 관점에서 평가해줘
```

## 핵심 팁

1. **데이터 형식**: 표 형태로 데이터를 제공하면 분석 품질이 높아집니다
2. **기준 명시**: "GAAP 기준", "K-IFRS 기준" 등 회계 기준을 명시하세요
3. **비교 기간**: 전분기, 전년 동기 등 비교 기간을 함께 제공하세요
4. **MCP 활용**: 실제 기업 데이터가 필요할 때 Alpha Vantage/Financial Datasets를 병행하세요
