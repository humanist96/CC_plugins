# 튜토리얼 03: 영업 플러그인 활용

> 난이도: 초급~중급 | 소요시간: 20분

## 학습 목표

- Sales 플러그인의 핵심 기능 이해하기
- 고객 리서치, 콜 준비, 아웃리치 작성 실습
- 일일 브리핑과 파이프라인 관리 실습

## Sales 플러그인 명령어 총정리

| 명령어 | 기능 |
|--------|------|
| `/sales:account-research` | 기업/인물 리서치 |
| `/sales:call-prep` | 미팅 준비 자료 |
| `/sales:daily-briefing` | 일일 영업 브리핑 |
| `/sales:draft-outreach` | 아웃리치 이메일 작성 |
| `/sales:pipeline-review` | 파이프라인 분석 |
| `/sales:forecast` | 매출 예측 |
| `/sales:call-summary` | 통화 요약 |
| `/sales:competitive-intelligence` | 경쟁사 분석 |
| `/sales:create-an-asset` | 영업 자료 생성 |

## 실습 1: 고객 리서치

### 시나리오
삼성SDS와의 미팅을 앞두고 있습니다. 기업 정보를 조사해봅시다.

### 1-1. 기업 리서치

```
/sales:account-research
```

프롬프트가 나타나면 다음과 같이 입력:

```
삼성SDS에 대해 리서치해줘. 최근 사업 동향, 주요 서비스, 경영진 정보 포함해서
```

### 1-2. 인물 리서치

```
삼성SDS의 CTO와 IT 인프라 담당 임원에 대해 조사해줘
```

## 실습 2: 미팅 준비

### 시나리오
내일 오후 2시에 삼성SDS 클라우드팀과 미팅이 잡혀 있습니다.

### 2-1. 콜 준비 자료 생성

```
/sales:call-prep
```

입력:

```
내일 삼성SDS 클라우드팀과 미팅이 있어.
참석자: 김철수 팀장, 박영희 과장
주제: 금융 클라우드 인프라 전환 프로젝트
우리 회사: 코스콤 (금융 IT 인프라 전문)
```

결과물에 포함되는 항목:
- 회사 배경 및 최근 동향
- 참석자 정보
- 추천 아젠다
- 핵심 질문 리스트
- 잠재적 이슈 및 대응 방안

### 2-2. 미팅 아젠다를 주가 데이터로 보강

```
삼성SDS(018260) 최근 주가 동향과 실적을 조사해서 미팅 준비 자료에 추가해줘
```

## 실습 3: 아웃리치 이메일 작성

### 시나리오
신한은행 디지털혁신팀에 처음 연락하려고 합니다.

### 3-1. 콜드 이메일 작성

```
/sales:draft-outreach
```

입력:

```
신한은행 디지털혁신팀에 첫 이메일을 보내려고 해.
우리 제품: AI 기반 금융 데이터 분석 플랫폼
타겟: IT 인프라/데이터 분석 담당자
핵심 가치: 실시간 시장 데이터 분석, 리스크 관리 자동화
```

### 3-2. 후속 이메일 작성

```
지난주 신한은행 미팅 후 팔로업 이메일을 작성해줘.
미팅에서 논의한 내용:
- 데이터 분석 플랫폼 데모 진행
- 보안 인증 관련 질문
- 다음 달 POC 일정 논의
```

## 실습 4: 일일 브리핑

### 4-1. 모닝 브리핑

```
/sales:daily-briefing
```

입력:

```
오늘 일정:
- 10:00 삼성SDS 팔로업 콜
- 14:00 KB국민은행 신규 미팅
- 16:00 팀 내부 파이프라인 리뷰

주요 관심사:
- 삼성SDS 프로젝트 계약 진행 상황
- KB국민은행 디지털 전환 예산 확보 여부
```

## 실습 5: 파이프라인 관리

### 5-1. 파이프라인 리뷰

```
/sales:pipeline-review
```

입력:

```
현재 파이프라인:
1. 삼성SDS - 클라우드 인프라 전환 - 50억 - 협상 단계 - 승률 60%
2. KB국민은행 - 데이터 분석 플랫폼 - 30억 - 발굴 단계 - 승률 20%
3. 하나은행 - API 게이트웨이 - 15억 - 제안 단계 - 승률 40%
4. NH투자증권 - 리스크 관리 시스템 - 25억 - 협상 단계 - 승률 70%
5. 신한은행 - AI 트레이딩 - 40억 - 발굴 단계 - 승률 15%
```

### 5-2. 매출 예측

```
/sales:forecast
```

위 파이프라인 데이터를 기반으로 분기별 매출 예측을 생성합니다.

## 실습 6: 통화 요약

### 6-1. 콜 노트 정리

```
/sales:call-summary
```

입력:

```
삼성SDS 미팅 노트:
- 참석자: 김철수 팀장, 박영희 과장, 우리측 이민수 부장
- 클라우드 전환 프로젝트 2분기 착수 희망
- 예산은 내부 승인 진행 중 (50억 규모)
- 보안 인증(ISMS-P) 보유 여부 확인 요청
- 경쟁사 A사도 제안 중이라 함
- 다음 주 기술 검토 미팅 일정 잡기로 함
- POC 환경 구성 요청
```

결과물:
- 핵심 요약
- 액션 아이템 리스트
- 팔로업 이메일 초안
- 내부 공유용 요약

## 핵심 팁

1. **구체적으로 입력**: 회사명, 담당자, 제품, 금액 등 구체적 정보를 제공할수록 결과가 좋습니다
2. **맥락 제공**: "우리 회사는 코스콤이고 금융 IT 인프라를 전문으로 합니다" 같은 배경 정보를 추가하세요
3. **단계적 활용**: 리서치 -> 콜 준비 -> 미팅 -> 콜 요약 -> 팔로업 순으로 활용하면 효과적입니다
